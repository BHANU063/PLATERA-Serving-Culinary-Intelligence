# Platera: Serving Culinary Intelligence

Platera is a modern web application that uses AI to help you create delicious meals from the ingredients you already have. With a beautiful, responsive interface and smart helpers, Platera is your personal kitchen assistant.

## Features

- **AI Recipe Generator:**
  - Enter a list of ingredients you have on hand.
  - Instantly receive creative, step-by-step recipes tailored to your items.

- **Ingredient Substitution Finder:**
  - Select any ingredient and get AI-powered suggestions for substitutes.

- **AI Sommelier:**
  - Get perfect wine pairings for your generated recipes.

- **Personal Recipe Box:**
  - Save your favorite recipes for easy access later.

## How It Works

1. **List Ingredients:**
   - Type in what you have in your fridge, pantry, or garden.
2. **Get AI Recipes:**
   - Click "Generate Recipes" to let the AI suggest meal ideas.
3. **Use AI Helpers:**
   - Find ingredient substitutes or wine pairings with a click.
4. **Save Recipes:**
   - Store your favorite recipes in your personal recipe box.

## Technologies Used
- **Frontend:**
  - HTML5, CSS3 (Tailwind CSS), JavaScript (ES6 modules)
- **AI Integration:**
  - Google Gemini API for recipe and suggestion generation
- **Authentication & Storage:**
  - Firebase (for user authentication and saving recipes)

## Setup & Usage
1. Clone or download the repository.
2. Make sure you have a valid Google Gemini API key and Firebase configuration.
3. Open `index.html` (or the `.dist/index.html` for production) in your browser, or serve with a local web server.
4. Enjoy creating and discovering new recipes!

## Notes
- This project is for educational and demonstration purposes. Do not expose API keys in production.
- For best results, use a modern browser.
